# nigelslibrary-site

The published site for **nigelslibrary.co.uk** — a durable digital record of the library of
Nigel William Minton (1948–2024), served via GitHub Pages.

**This repo holds generated output only.** Every page here is a disposable, regenerable view:
the catalogue, the generators, and the family's working records live in a separate private
repository. If this site is ever lost, it can be rebuilt in full from that source — nothing
here is the original of anything.

For the future maintainer: the one thing that must never lapse is the **domain registration**
(`nigelslibrary.co.uk`, registered to 2036). The `CNAME` file in this repo binds GitHub Pages
to it; keep that file present in every deploy.
